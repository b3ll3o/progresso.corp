# API E2E Tests

Este projeto contém os testes end-to-end (E2E) para a API do sistema ProgressoCorp.

## Executando os Testes

Para rodar os testes E2E da API, utilize o comando:

```bash
nx e2e api-e2e
```

Certifique-se de que os serviços de infraestrutura (Banco de Dados, Redis) estejam rodando.
