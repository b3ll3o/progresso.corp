# Frontend E2E Tests

Este projeto contém os testes end-to-end (E2E) para o Frontend do sistema ProgressoCorp, utilizando Playwright.

## Executando os Testes

Para rodar os testes E2E do Frontend, utilize o comando:

```bash
nx e2e frontend-e2e
```

Certifique-se de que a API e o Frontend estejam acessíveis ou configurados para serem iniciados pelo teste.
